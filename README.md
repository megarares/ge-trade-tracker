# GE Trade Tracker Plugin

GE Trade Tracker is a RuneLite plugin that captures your Grand Exchange offer snapshots and sends them to the GE Trade Tracker API for tracking and analysis.

## What it does

- Monitors Grand Exchange offer slot changes while you are logged in.
- Collects offer state details like item ID, price, quantity, and progress.
- Sends offer snapshots (with player/account identifiers) to the configured API endpoint.
- Skips duplicate payloads and debounces rapid updates to reduce unnecessary requests.
